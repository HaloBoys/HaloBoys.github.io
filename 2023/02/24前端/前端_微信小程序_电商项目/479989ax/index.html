项目配置

配置网络请求

由于平台的限制，小程序项目中不支持 axios，而且原生的 wx.request() API 功能较为简单，不支持拦截器等全局定制的功能。因此，建议在 uni-app 项目中使用 @escook/request-miniprogram 第三方包发起网络数据请求。@escook/request-miniprogram 官方文档

  1. 如果项目中没有 package.json 文件，需要使用 npm init -y 命令初始化一个
  2. 在命令行输入：@escook/request-miniprogram 安装
