Windows C 盘清理

发表于2023-02-11|更新于2024-04-07|折腾性能优化
|字数总计:459|阅读时长:1分钟

清理缓存文件

  1. Windows +R打开运行，然后输入 %temp%
  2. 打开临时文件夹，这里的文件都是可以清理的
  3. Ctrl+A全选，Shift+delete直接永久删除

磁盘清理

  1. 在我的电脑中，选中C盘，右键 —> 属性，点击磁盘清理
  2. 这里的文件都是可以直接删除的，全选—> 直接删除即可

休眠文件

hiberfil 休眠文件，简单来说就是，你关机时会将所有内存中的数据，写入到硬盘，然后再关闭系统，在下次启动开机时，将保持的数据在写回内存。
被保存的数据就是休眠文件，笔记本用户可能会用到此功能。但拥有同样功能的睡眠启动更快（就是不能关闭系统。）

关闭方式：

  1. 管理员身份打开运行
  2. 输入powercfg -h off 指令即可关闭休眠文件
  3. 如果后面又要用到休眠功能，只需要将 off 改为 on 即可再次打开该功能，没有任何损失。

不要将软件安装到C盘

很多软件在安装时，会默认安装至C盘。

建议在其他盘符（C盘以外）中，安装软件，因为软件一般会在当前目录中生成缓存文件。

安装至其他盘，可大大降低C盘爆满速度！

清理临时文件

  1. WIN + I 快捷键打开设置在设置中搜索存储找到临时文件选项
  2. 这里的文件都是可以清理的，删除文件即可

清理软件缓存

很多软件默认将缓存保存在C盘，毫无疑问，大大加快了C盘的爆满速度，这里微信为例。

  1. 找到微信文件管理界面
  2. 点击更改，选择一个非C盘的路径。
