金蝶云星空服务器IP固定

物料中的字段审核过后可编辑修改

如编辑物料的采购页签最小包装量审核后可修改（不用工程那边反审核），其他单据都是一样的操作

  1. BOS 中打开业务对象，选择对应子对象
  2. 菜单栏选择编辑 -> 批量编辑字段属性
  3. 选择已审核取消勾选需要被编辑的字段

设置字段为过滤筛选条件

如设置物料已使用字段作为筛选条件

  1. 选择物料业务对象
  2. 菜单栏选择编辑 -> 批量编辑字段属性
  3. 选择对应字段，设置可见性，列表可见性中的选项勾选：

批量批改字段自定义

在对应模块选择需要被批改的字段，选择：功能控制 —> 允许委外

设置点位累计添加方法

新增行和删除行都会触发汇总计算

  1. BOS 中打开业务对象，选择对应子对象
  2. 添加小数字段（勾选隐藏尾零选项可在文本框没有获得焦点时不显示0）
  3. 菜单栏选择编辑 -> 批量编辑字段属性 -> 子项明细 -> 点位
  4. 汇总至上级子段（第二步添加的小数字段）

点击父级分类显示子级内容

默认只能点击到最后一级分类才显示分类中的内容，取消勾选此选项即可点击父级分类显示子级分类内容

添加旧订单编号字段并可关联下推至下游单据

实现从老系统没有结束迁移补单过来的单据，需要补充老系统的旧订单号

以：采购订单 —> 采购入库 下推旧订单号自动从采购订单到采购入库 为例：

  1. 采购订单和采购入库都添加文本字段修改标题为旧订单号（必须都添加，不然下面字段映射关系中找不到）
  2. 选择单据转换，选择采购订单 --> 采购入库

  3. 选择拓展（停用）选择字段映射关系，给单据都绑定上旧订单号字段

给单据添加旧物料编码字段（给任意单据添加系统没有内置的字段）

如：给物料清单的父项和子项添加上旧物料编码字段

  1. 在BOS中选择物料清单单据

修改单据默认编码规则

  1. 在编码规则列表中自定义需要的编码规则，设置成默认
  2. 在单据类型列表中指定单据的编码规则

新增单据时单据编号字段可编辑

  1. 【不确定需不需要操作第一步】
    在BOS平台中选中对应单据，设置单据编号字段的锁定性
  2. 使用Administrator用户登陆客户端，在单据参数配置中取消勾选此选项
  3. 在对应单据的编码规则中，勾选手动补号和自动补号
