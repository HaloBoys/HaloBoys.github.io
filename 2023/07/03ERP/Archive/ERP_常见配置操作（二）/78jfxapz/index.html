委外领料相关步骤

  1. 先打印单据，给委外供应商，实际发多少后面再做调拨到委外仓
  2. 调拨到委外仓之后再审核领料单，扣料

取消单据体不能使用块粘贴

生产/委外变更用料清单

  1. 打开生产/委外用料清单变更单新增，然后点击单据体中的变更或替换按钮，选择对应的用料清单中的物料。
  2. 返回数据，填写需要变更的物料即可（被替换的物料不会被删除，而是应发数量变成了0）

场景1：生产领料领了一半需要变更另一款物料

生产领料领了一半需要变更另一款物料，另一款物料的数量等于第一款物料未领的数量

  1. 新增生产用料清单变更单，替换，选择对应的用料清单，选择需要替换的物料
  2. 默认替换的新物料应发数量等于被替换物料的应发数量，通过调整被替换的新物料的分子来控制应发数量。分子 = 未领数量 / 应发数量

委外订单领料后变更供应商

  1. 委外订单对应的用料清单
  2. 委外订单对应的采购订单
  3. 委外订单对应的委外领料单

委外超耗单应用场景

在委托供应商加工的过程中，可能会存在由于供应商加工原因导致部分原材料损坏，这部分因供应商责任造成消耗的材料超出工艺定额，我们需要将损耗材料重新发给供应商并收取材料款，此种业务可以通过委外超耗单处理。

  • 超耗单 = 领料单 + 应付单（负数）
  • 直接打印超耗单发料给供应商
  • 超耗单审核过后需要下推应收单，填写原材料单价
  • 下推应收单必须选择对应的客户，且需要在客户选择中维护对应的供应商（加工商）

委外订单中的备注同步到委外用料清单【待处理】

方案一：在委外订单中查看对应的备注，复制委外订单的单据编号，在用料清单中按委外订单编号字段筛选

委外订单申请中查看是否下推采购订单（委外）

在委外订单申请中过滤出 采购订单 字段，下推过的采购订单会自动带出来是哪个采购订单。

委外订单申请领料一半需要终止

  1. 变更采购订单（委外）的数量
  2. 变更委外订单申请的数量（使用委外订单变更单去变更）

变更单生效之后委外订单申请对应的委外用料清单会自动更新

采购价目表阶梯价格

采购价目表阶梯价格

注意事项：

  1. 左开右闭规则。有效区间为大于从字段，小于等于至字段。
    1. 如果【从】字段全部为 0 的话，跟以前没有从字段一样的效果。【如果需要包含从的值，可以去掉从字段的值，只设置至字段的区间】
    2. 如果只设置 1000 以上才取价目表的话则只设置从的值为999即可
  2. 采购价目表通过采购调价表来修改价格和新增物料价格
  3. 被采购订单引用了可以删除吗？【可以禁用】
  4. 采购价目表的生效日期之前的单据不会取到价格

拓展：

  1. 可以通过 货源清单 设置物料只允许通过哪些供应商来采购
  2. 可以在物料中设置默认的供应商（通过采购申请下推采购订单自动带出来供应商）
  3. 采购订单中可以业务查询：
    1. 价格查询
    2. 历史价格
  4. 强制以采购价目表的价格为准：
    • 在采购管理系统参数 价格管理中设置采购订单的条件：FBusinessType ='CG' 为 强制价目表定价
  5. 可以添加相同物料相同供应商的价目表，但是生效日期不能一样，后生效的会覆盖之前生效的价目表

待测试：

  • [x] 价目表不设置供应商会有什么效果？
    • [x] 不设置供应商，在采购订单不填写供应商，只填写物料和数量即可带出价格，相当于是一个公共价目表
  • [x] 价目表和供应商都设置价目表，供应商的价目表权重大于公共价目表
  • [x] 供应商设置公共价目表为默认价目表，与供应商的价目表权重谁大？
    • [x] 供应商的价目表权重大
  • [x] 先不录入阶梯价，后续再维护可不可行？
    • [x] 可以，直接选中对应的行，然后插入行
  • [x] 采购限价控制强度设置禁止交易后，公共价目表和供应商价目表的的价格都可以取吗？
    • [x] 公共价目表和供应商价目表都存在时，只能取供应商价目表的价格，不能取公共价目表价格
    • [x] 只有公共价目表时，可以取公共价目表价格
  • [x] 采购价目表调价？调价后之前的历史价格如何呈现？
    • [x] 可以呈现，呈现的过程调价的这个动作，每次调价都会有调价后的价格

采购订单只能从上游单据下推

在单据类型中设置源单编号必录即可（必须有上游单据）

单据作废的应用场景

某张单据可能由于操作错误或者下单重复等原因需要重新做单据,此时可以在列表直接点击删除,删除之后这张单据后不会再在系统中体现。

而作废的操作的意义是,企业对制单要求比较严格,不允许对单据进行随意删除,即使单据错误不需要再继续进行也需要在系统中有记录等情况时,可以使用作废;

单据是创建或者重新审核的状态下才可以作废,系统控制了作废状态的单据是不允许删除的,作废的意义是切断该单据的关联性,保留单据数据用来记录,不会在其他报表中体现,也没有其他影响。

既是供应商又是客户

既是供应商又是客户：新增客户的时候要在基本信息->相关->选择对应供应商，最后结算的时候会在财务会计应收转销模块中进行冲销结算。

部分领料入不了库

有部分领料的情况，入库入不了情况：
方式1. 在单据类型中选择生产订单，关闭生产入库材料领用控制的严格控制
方式2. 在用料清单中单独控制某款物料

单据关闭状态和业务关闭
