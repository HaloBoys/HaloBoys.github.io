|字数总计:922|阅读时长:3分钟

VueRouter

官方文档链接：https://router.vuejs.org/zh/installation.html

Install

Use

Redirect 重定向

嵌套路由（子路由）

当页面中组件中需要嵌套子组件的时候，可以使用路由的嵌套路由特性

动态路由

动态路由将 Hash 地址中可变的部分定义为参数项，提高路由的可复用性

$route.params.id 获取参数

组件中可以直接通过 [this.]$route.params.id 获取当前跳转的动态路径 “id”

拓展

hash 地址中 / 后面的参数项叫做”路径参数”，可以通过 this.$route.params 访问

hash 地址中 ?后面的参数项叫做”查询参数”，可以通过 this.$route.query 访问

this.$route 中，path 只是路径的部分，fullPath 才是完整的地址

props 传参

在路由规则中也可以通过添加 props: true 属性来传递参数

编程式导航跳转

点击链接（ ）实现导航的方式叫做声明式导航，通过调用API方法实现跳转的方式叫做编程式导航

vue-router 常用编程式导航 API：

  1. this.$router.push(‘hash地址’)
    • 跳转指定的hash地址，增加一条历史记录
  2. this.$router.replace(‘hash地址’)
    • 跳转指定的hash地址，替换掉当前的历史记录
  3. this.$router.go(数值n)
    • 在浏览历史中前进或后退指定步数，负数后退，正数前进
    • this.$router.go 简写
      • $router.back()
      • $router.forward()

路由导航守卫

导航守卫可以控制路由的访问权限

全局前置守卫

每次路由跳转时，都会触发全局前置守卫，因此在全局前置守卫中可以对每个路由进行访问权限控制

router.beforeEach((to, from, next)：

  • to: Route: 即将要进入的目标 路由对象
  • from: Route: 当前导航正要离开的路由
  • next: Function: 一定要调用该方法来 resolve 这个钩子。执行效果依赖 next 方法的调用参数。
    • next() 放行
    • next(false) 中断当前的导航
    • next('/') 跳转地址
