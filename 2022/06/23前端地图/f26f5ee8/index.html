前端地图

发表于2022-06-23|更新于2022-06-28|前端
|字数总计:927|阅读时长:3分钟

目标: 打好基础，找一个5K+ 的工作，切勿眼高手低，走马观花。要做到一步一个脚印，从最基础的开始做起。

前端基础

前端最基础的部分，多花时间，掌握好每一个知识点。

HTML

  • ✅ 学习 HTML 基础，标签、元素、表单验证等等 【HTML5语义化标签 HTML新增标签属性 表单项新增属性】
  • ✳️ 语义化标签
  • ✳️ 了解 Web 无障碍（Accessibility）残疾人
  • ✳️ 学习 SEO 基础的优化

CSS

  • ✅ 学习 CSS 基础
  • ✅ 制作布局：浮动、定位、显示、盒模型、网格布局、弹性布局
  • ✅ 响应式设计和媒体查询（@media）
  • ✅ 结合 HTML 制作一个简单的网页作为最佳实践

CSS选择器

CSS动画Animation

经典布局练习（双飞翼，品字布局）+案例
响应式布局

JavaScript

  • ✅ 学习语法和基本结构
  • ✅ 学习操作 DOM
  • ✅ 学习 Fetch API / Ajax（XHR）/ axios
  • ✅ ES6+ 和模块化 JavaScript
  • ✅ 了解变量提升、事件冒泡机制、作用域、原型、Shadow DOM、严格模式等概念
