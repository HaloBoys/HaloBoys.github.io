uToos 为生产力而生

发表于2022-05-21|更新于2023-08-03|工具Software
|字数总计:878|阅读时长:2分钟

utools 这种插件化的管理方式，为用户带来了很大的自由度，能够帮助你打造出更趁手的私人专属工具库。个人觉得这是一款非常好用的软件 工作效率直接+100%，免费版本功能也完全够用，希望越来越多的人能支持这种国产优秀软件。

安装

utools 官网：https://u.tools/ 选择对应系统版本安装包安装。

一台电脑个人觉得免费版本足够使用了，如果需要多端数据同步，也可以开通 uTools会员，更多会员权益自行查看。

使用

绑定好账号之后就可以使用了，直接在插件应用市场选择想要的插件安装即可。

软件快捷键

以下快捷键均可在偏好设置中自定义。

  1. Alt + Space 显示/隐藏搜索面板，可快速启动已安装的应用等
  2. Ctrl + D 分离窗口快捷键
  3. 长按鼠标中键 呼出超级面板

自定义插件快捷键

给常用的插件自定义快捷键可快速呼出插件，注意快捷键冲突问题。

插件推荐

这里推荐几个实用的插件

讯飞 ocr

讯飞 ocr 是一款截图识字的插件，一般来说这种识图api接口都是收费的，目前插件内有很多共享的接口供使用。个人使用了一段时间感觉挺不错的，这款插件搭配另一款聚合翻译插件可在截完图之后直接翻译识取内容。

PDF 转换器

将 PDF 转换为多种格式

批量重命名

导入想要重命名的图片，自定义命名规则。

图片批量处理

相见恨晚！

我的插件

图床

无意间发现的这款插件，还不错，支持大部分图床仓库

安装插件

首先在插件应用市场安装图床插件。(没错，插件名就叫“图床”)

安装完成后可自行配置图床仓库。

创建仓库(Gitee)

大型打脸现场，刚夸了夸Gitee，这就图片加了防盗链，不能直接访问了，换别的图床吧，想白嫖果然不容易。

注册 gitee 账号，创建仓库，仓库名称随意，建议：utools-filebed 创建记得初始化仓库，并开源 (图床的仓库一定要开源)。

申请私人令牌

  1. 地址： https://gitee.com/profile/personal_access_tokens
  2. 点击”生成新令牌”，令牌描述随意取，推荐utools-filebed，权限请勾选”projects”
  3. 提交后务必保管好生成的令牌，此令牌只显示一次。

配置插件

进入图床插件：

  1. 将刚刚申请到的 token 填 access_token
  2. 用户名是gitee的用户名
  3. repo 填仓库名称 (不带用户名)
  4. path 就是图片上传完成之后的命名格式，建议填写：{Y}-{M}-{D}{H}:{m}:{s}- 输出为：2021-9-1718:06:09-xxx.png

切换

填写完成后在图床插件这里选择 gitee 即可

[可选]

如果想搭配 Typora 进行上传的话需要安装 utools图床插件命令工具

使用Typora上传或使用命令行工具期间保证图床插件的运行, 不能完全退出图床插件,不能在「utools」主输入窗口输入「clear 」命令。此外需要环境版本 node >= 12
