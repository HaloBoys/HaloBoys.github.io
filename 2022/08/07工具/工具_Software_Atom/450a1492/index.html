安装 Atom Plugin 需要本地安装 Node + git 环境，这里使用命令行来安装 Atom 插件（Atom提供的使用 apm install 指令的安装方式依然是依赖于npm）

语法格式

apm install [packages]

插件安装路径

C:\Users\<UserName>\.atom\packages

Sync-Settings

sync-settings 是一款备份插件，可以备份ATOM的全局设置、插件、按键绑定(keymaps)、界面样式、代码片段(snippets )、 init script。有了这项功能以后再次恢复插件就十分方便。

配置

安装 sync-settings

  1. Ctrl + ,
  2. 安装

配置sync-settings

  1. 打开Setting可以看到我们需要配置 GistID 和 token
  2. 打开 Github 创建 GistID 和 token
    1. 创建 GistID
      1. https://gist.github.com/
      2. 新建 gist
      3. 保存地址栏的 GistID
    2. 创建 token
      1. Settings —> Developer settings —> Personal access tokens —> Generate new token
      2. 填写 token 名称，勾上gist权限。
  3. 将 GistID 和 token 填入即可。

备份与还原

备份

备份命令：

  1. Ctrl + shift + P 打开命令面板
  2. 输入 Sync Settings: Backup

备份成功

还原

  1. Ctrl + shift + P 打开命令面板
  2. 输入 Sync Settings: Restore
